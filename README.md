Description

A Java-based console application that simulates a vehicle showroom where customers can browse and purchase available vehicles. Users can select between 2-wheelers and 4-wheelers, choose specific models, brands, and colors, view detailed vehicle information, and generate a final bill during checkout.

Features

Display available vehicles
Separate categories for 2-Wheeler and 4-Wheeler
View bikes, scooters, and cars
Select vehicle by model, brand, and color
Display detailed vehicle specifications
Purchase multiple vehicles
Checkout and bill generation
Menu-driven console application

Console-based UI
